React-Naitve学习总结

发表于 2020-06-23 分类于 React-Native
本文字数： 2.1k 阅读时长 ≈ 2 分钟

React Native 结合了 Web 应用和 Native 应用的优势，可以使用 JavaScript 来开发 iOS 和 Android 原生应用。在 JavaScript 中用 React 抽象操作系统原生的 UI 组件，代替 DOM 元素来渲染等。

React Native 使你能够使用基于 JavaScript 和 React 一致的开发体验在本地平台上构建世界一流的应用程序体验。React Native 把重点放在所有开发人员关心的平台的开发效率上——开发者只需学习一种语言就能轻易为任何平台高效地编写代码。Facebook 在多个应用程序产品中使用了 React Native，并将继续为 React Native 投资。

本项目汇集了各类 React Native 学习资料、工具、组件、开源 App、以及相关新闻等。

组件

UI

react-native-elements：一款优秀的 UI 组件工具
NativeBase：一款为 React Native 提供的 UI 组件
react-native-material-design：一款用于 React Native 上的材料设计 UI 组件库
react-native-vector-icons：用于 React Native 的可定制的图标资源库，支持 NavBar/TabBar, image source 以及样式

Navigation

react-navigation：最新最流行的导航组件
react-native-tab-navigator：一款兼容 Android、iOS 的 TabBar组件

ViewPager

react-native-swiper：一款轮番滑动的组件
react-native-looped-carousel：一款滚动轮播组件

ListView & ScrollView

react-native-refreshable-listview：下拉刷新组件
react-native-swipe-list-view：滑动删除组件
react-native-swipeout：iOS 样式的划动删除组件
react-native-sortable-listview：拖拽排序组件
react-native-SortableList：拖拽排序组件

Text & Rich Content

react-native-htmlview：HTML 显示组件，渲染 HTML text

TabLayout

react-native-scrollable-tab-view：一款用于 React Native 上 TabLayout 组件

弹框

react-native-easy-toast：一款用于 React Native 上消息提示弹框组件，使用方便，支持定义 Toast，支持 iOS，Android
react-native-popover：一款类似 Android popupwindow 的弹出框组件

其它

tcomb-form-native：强大的表单处理控件，支持 JSON 模式，可插拔的外观和感觉
